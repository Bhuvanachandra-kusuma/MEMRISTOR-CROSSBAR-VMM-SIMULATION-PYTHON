# MEMRISTOR-CROSSBAR-VMM-SIMULATION-PYTHON


This project simulates a 4×4 memristor crossbar performing analog vector–matrix multiplication (VMM), the fundamental computation used in neuromorphic hardware and in-memory AI accelerators.

It implements:
- A nonlinear HP-type memristor model
- Programmable conductance matrix
- Analog VMM computation (G × X)
- Device variability and noise modeling
- Conductance heatmap and output comparison

 Tools Used
- Python
- NumPy
- Matplotlib

 Features
- Memristor nonlinear drift model  
- Crossbar array simulation  
- Ideal vs analog VMM comparison  
- Noise + variability effects  
- Heatmap visualization
